# Pizza Sales Analytics Dashboard (Power BI)

This project presents an interactive Power BI dashboard analyzing pizza sales performance across categories, sizes, and time. The objective is to identify sales patterns, top and bottom performers, and key business insights.

## Business Questions

- What is overall sales performance?
- Which categories and sizes contribute most to revenue?
- What are the daily and monthly order trends?
- Which pizzas are top and bottom performers?
- When are peak sales periods?

## Key Features

- KPI cards for revenue, orders, and averages
- Daily and monthly trend analysis
- Category and size contribution
- Top and bottom 5 pizzas by revenue, quantity, and orders
- Multi-page navigation
- Interactive filters

## Tools Used

- Power BI  
- DAX  
- Data modeling
